# 📒 Notebooks

This folder contains the Jupyter notebooks used during the development of **ResearchMind AI**.

Each notebook represents a different stage of the project pipeline.

---

## 📚 Notebook Overview

### 01_Data_Preparation.ipynb

- Load the research paper dataset
- Clean and preprocess the data
- Save the cleaned dataset

---

### 02_PDF_Processing.ipynb

- Extract text from PDF research papers
- Perform text preprocessing
- Prepare documents for analysis

---

### 03_Semantic_Search.ipynb

- Generate sentence embeddings
- Build a FAISS vector index
- Perform semantic similarity search

---

### 04_Research_Assistant.ipynb

- Generate paper summaries
- Extract keywords
- Analyze research papers
- Generate citations

---

### 05_Final_Demo.ipynb

Demonstrates the complete ResearchMind AI workflow from PDF upload to report generation.

---

## 🎯 Purpose

These notebooks were primarily used for:

- Development
- Experimentation
- Testing
- Model evaluation

The final production-ready application is available through:

```bash
python main.py
```
