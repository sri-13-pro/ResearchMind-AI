# 📄 Uploads

This folder contains research papers (PDF files) used for analysis.

---

## Supported Format

- PDF (.pdf)

---

## Example

```
uploads/

attention_is_all_you_need.pdf

bert.pdf

gpt.pdf
```

---

## Usage

Place one or more research papers inside this folder.

When the application starts, it automatically lists all available PDF files for analysis.

```bash
python main.py
```

Select the desired paper from the menu to begin analysis.
