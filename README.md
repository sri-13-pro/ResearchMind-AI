# 📖 Overview

ResearchMind AI is an intelligent research paper analysis system developed to help students, researchers, and developers understand academic papers quickly using Artificial Intelligence and Natural Language Processing.

Instead of manually reading lengthy research papers, the system automatically extracts text, generates summaries, identifies keywords, performs semantic search, visualizes paper analytics, and generates professional reports.

---

# ✨ Features

## 📄 PDF Processing

- Extract text from research papers
- Supports academic PDF documents
- Clean and preprocess extracted text
- Automatic text extraction using PyMuPDF

---

## 🤖 AI Summarization

Generate concise summaries of long research papers using NLP techniques.

Features:

- Extractive summarization
- Research overview generation
- Fast paper understanding

---

## 🔑 Keyword Extraction

Automatically identifies the most important research keywords.

Example:

- Transformer
- Attention
- Encoder
- Decoder
- Neural Networks
- Language Models

---

## 🔍 Semantic Search

Powered by

- Sentence Transformers
- FAISS Vector Database

Search research papers using natural language.

Example:

```text
Transformer architectures for NLP
```

Returns the most semantically similar research papers.

---

## 📊 Research Paper Analytics

Generate insights including:

- Character Count
- Word Count
- Unique Words
- Reading Time
- Word Frequency
- Word Cloud
- Frequency Charts

---

## ☁️ Word Cloud

Automatically generates a Word Cloud showing the most important words.

Output


## 📈 Frequency Analysis

Creates a frequency chart of the most common words.

Generate citations in multiple academic formats.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/sri-13-pro/ResearchMind-AI.git
```

Navigate into the project

```bash
cd ResearchMind-AI
```

Create a virtual environment

```bash
python -m venv venv
```

Activate the environment

### Windows

```bash
venv\Scripts\activate
```

### Linux / macOS

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```
